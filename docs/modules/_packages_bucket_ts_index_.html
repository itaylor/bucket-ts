getBucketProvider

  • getBucketProvider(name: string, options: BucketProviderOptions): BucketProviderExt
    • Defined in packages/bucket-ts/index.ts:35

    Parameters

    • name: string

      the name of the BucketProvider instance to fetch

    • options: BucketProviderOptions

      the options for the BucketProvider. This will be validated against its JSON Schema at runtime. Also, the options will be processed for environment variable substitution. Example:

      getBucketProvider('s3', {
  accessSecretId: 'mySecretId',
  secretAccessKey: '${{ MY_SECRET_KEY }}',
});

      would place the value of process.env.MY_SECRET_KEY into the secretAccessKey value. Note: options must be serializable to JSON.

    Returns BucketProviderExt

    a new BucketProviderExt instance for the provider
